API Endpoints

Use these endpoints to interact with our dataset. All responses are returned in JSON format.

GET Endpoints

Endpoint Description Query Params Success Code
/getAll Returns the entire list of items. None 200
/getById Returns a specific item by name (case-insensitive). name (required) 200
/search Filters items by name and/or region (case-insensitive). name (optional), region (optional) 200
/meta Returns the total count of items. None 200

POST Endpoints

Note: Content-Type must be application/x-www-form-urlencoded or application/json.

Endpoint Description Body Fields Success Code
/addItem Creates a new item. name (required),
capital (required),
region (required),
currencyName (optional),
currencySymbol (optional) 		201
/editItem Updates an existing item by name. name (required, current name),
newName (optional),
capital (optional),
region (optional),
currencyName (optional),
currencySymbol (optional) 		200
